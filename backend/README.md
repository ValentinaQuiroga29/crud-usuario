# Backend de Usuarios (Node.js + Express)

## Requisitos
- Node.js >= 14
- npm

## Instalación

```bash
cd backend
npm install
```

## Ejecución

### Modo desarrollo (con recarga automática):
```bash
npm run dev
```

### Modo producción:
```bash
npm start
```

## Endpoint de ejemplo
- `GET /usuarios` — Devuelve una lista de usuarios de ejemplo.
